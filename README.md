------
node.js  
mongodb

クライアント側
------
backbone.js  
require.js

動かし方
------
mongodb起動 → node.js起動

coffeescriptのまま起動するのであれば以下

    # coffee app.coffee

ブラウザ上で、以下のURLを入力する
http://localhost:3000/appsettings/1
